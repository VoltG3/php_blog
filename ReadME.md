## LTPC_Blog
### HTML / PHP / CRUD
<br>

### Requires
 - Localhost  
 - MySql
 - phpMyAdmin

### CRUD scripts
Run this script to auto create and fill up database for test purpose
```
./php/MySQL_create_DB.php   
```

#### creation sequence
```
./php/MySQL_create_DB.php       - first
./php/MySQL_create_TABLE.php    - next call
./php/MySQL_insert_DATA.php     - next call

```

#### connection details
```
./config.php
```
